# cryptoip
> VOIP Client / Server in nodeJs(electron) encrypted in AES
#### Current version 1.0

### Requirement:

#### Server
- express
- http
- socket.io
#### Client
- electron

### Features:
- Real time voip
- Encrypted packets
- GUI with electron JS
- Server can handle mutliple clients
- Good voice quality
