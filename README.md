- http

- electron
- crypto

### Features:
- Real time voip
- Encrypted packets
- GUI with electron JS
- Server can handle mutliple clients
- Good voice quality
- Chat encrypted with AES 128
- Voice stream encrypted with AES 128
- Private messages
- Unique username
- Signature check
